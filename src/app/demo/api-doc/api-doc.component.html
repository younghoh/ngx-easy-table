API Documentation

Table has functionality to control it's content outside component. It means by using API you can control table's behaviour as you want.

To define API object use code snippet defined below:

Type Code Returns Description
API.setInputValue
void Set input value. You can set array of values.
API.rowContextMenuClicked
void Emit event to close context menu
API.setRowClass
void Set table row class
API.setCellClass
void Set table cell class
API.setRowStyle
void Set table row style
API.setCellStyle
void Set table cell style
API.setTableClass
void Set table class (for example Bootstrap)
API.setPaginationCurrentPage
void Set pagination on given page
API.getPaginationTotalItems
void Get pagination total items
API.getPaginationCurrentPage
void Get pagination current page
API.sortBy
void Set sort by column
